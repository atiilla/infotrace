## Development

### Requirement

- Python 3.8+

### Installation

$ `git clone git@github.com:atiilla/infotrace.git`

$ `cd infotrace`

$ `pip install -r requirements.txt`

$ `streamlit run app.py`

Streamlit will be served at http://localhost:8501
